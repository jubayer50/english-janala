## English Janala is an English vocabulary website.

This is an amazing project. All data comes through API fetch and displaying this information in the UI (fully dynamically). There are some lesson buttons. It also came from the API when someone clicks on the lesson’s button, then the lesson's vocabulary will show on screen. when you click on the vocabulary details icon, more information about the vocabulary will show on popup. Besides the details icon, there is another sound icon. When someone clicks on that sound button, the pronunciation of the vocabulary will be heard, which is also an amazing feature of this project.

---

## The technologies used in this project are:

- JavaScript(vanilla)
- The most popular CSS framework TailwindCSS, and DaisyUI.

---

## Three features of this project:

- More information will show on popup, click on the details icon
- Correct pronunciation of words, click on the sound icon
- Find vocabulary by searching
